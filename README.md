<h2>Patrones de diseño que se han implementado en el proyecto:</h2>
<ul>
  <li><b>Singleton</b>: garantiza que siempre se obtenga la misma instancia del tipo solicitado, mientras que el patrón Factory generalmente proporciona una instancia diferente de cada tipo</li>
  <li><b>Front-Controller</b>: centraliza y estandariza el proceso de manejo de solicitudes. Implementa los controllers del programa.</li>
  <li><b>Connection Pool</b>: crea las conexiones de fomra segura y luego las baja, evitando cualquier problema que pueda conllevar.</li>
</ul>
